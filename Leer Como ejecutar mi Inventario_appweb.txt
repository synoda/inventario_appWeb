paso 1 - Para ejecutar esta aplicación web se debe de descomprimir la carpeta de inventario_appWeb
paso 2- Abrir una terminal de CMD desde la ubicación del proyecto
paso 3- Instalar las librerías del proyecto para que pueda funcionar

**Librerías a instalar** 
pip install flask
pip install flask-sqlalchemy
pip install flask-migrate
pip install flask-wtf
pip install flask-login
pip install wtforms
pip install email-validator
pip install werkzeug

Paso 4 Ejecutar el comando:  Flask run 

cuando ejecute el comando Flask run ingrese en el link del servidor que sirve flask e ingrese con las claves del admin
